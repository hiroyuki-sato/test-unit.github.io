A Capybara 2.1.0 support release.

Improvments

1.0.3 – 2012-11-29

A support Capybara 2.0.1 release.

Improvments

  • Supported Capybara 2.0.1. It requires Capybara >= 2.0.1 and test-unit >= 2.5.3. Notice: Capybara 1.X aren’t supported yet from this release.

  • Added Test::Unit::Capybara::Assertions#assert_not_find.
  • Supported Capybara::ElementNotFound as a failure.
